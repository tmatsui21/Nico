Nico
======

## Description
iOS App dedicated to Maltese "Nico" for learning SwiftUI

## Requirement

For talkWithNico view
- install Gemini SDK
  - https://github.com/google/generative-ai-swift
- store API key in GenerativeAI-Info.plist file
